### Description:

Create a function that takes an integer as an argument and returns "Even" for even numbers or "Odd" for odd numbers.

#### C 

```C
const char* even_or_odd(int number)
{
  if (number == 0) return "Even";
  return number & 0x01 ? "Odd" : "Even";
```

```C
const char* even_or_odd(int number)
{
  static long long r;

  return (const char *)&r;
```
